# Drumtrack.js

A simple 4 sample drum machine using web audio buffers for performance. Has a sample for kick, snare, open hi hat and closed hi hat.

### Technologies used:

1. Web Audio API
2. Materialize CSS
3. Surge deployment

[Drumtrack.js](http://mstanton98-drumtrackjs.surge.sh/)
